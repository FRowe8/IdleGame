Paradox Protocol is a unique idle game where you manipulate time itself to generate resources through temporal arbitrage.

Core Mechanics:

  • Causal Loops: Send resources back in time to create compound interest cycles
  • Paradox Risk: Too many loops increase instability
  • Time Bank: Store credits with exponential returns
  • Offline Progress: Game simulates up to 24+ hours while you're away

Controls:

  • Mouse: Click UI buttons and interact with timeline
  • ESC: Pause/Menu (desktop only)
